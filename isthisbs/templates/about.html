About IsThisBS

What this is

IsThisBS is an automated fact-check publication. Every article is a single claim, checked against independent sources, with the receipts to show for it. There is no house opinion and no editorial spin: the site presents verdicts produced by Lenz and links to the full analysis for each one.

How the checks work

Each claim is verified by Lenz, an independent fact-checking engine. Several frontier AI models independently research the claim against fresh, external sources and assess it on the evidence. Where the models disagree, that disagreement is surfaced rather than hidden — an article marked panel divided is one the panel did not settle unanimously. IsThisBS adds no separate moderation layer; it is the reading experience on top of those results.

What the meter means

The BS Meter is dual-labeled: a plain-English BS reading alongside the canonical verdict. Color is never the only signal — the label is always written out.

Built on the Lenz API

This whole site is a small, public, MIT-licensed Python program that reads the public Lenz catalog through the lenz-io SDK and renders static HTML — no server, no database of its own. It doubles as a reference implementation for anyone building on the API.
