About IsThisBS

What this is

IsThisBS is an automated fact-check publication. Every article is a single claim, checked against independent sources, with the receipts to show for it. There is no house opinion and no editorial spin: the site presents verdicts produced by Lenz and links to the full analysis for each one.

How the checks work

Each claim is verified by Lenz, an independent fact-checking engine. Several frontier AI models independently research the claim against fresh, external sources and assess it on the evidence.

What the meter means

The BS Meter is dual-labeled: a plain-English BS reading alongside the canonical verdict. Color is never the only signal — the label is always written out.

The desk on X

The desk answers summons on X as @isthisbs. Three steps:

  1. See a post making a claim that smells off.
  2. Reply to it and tag @isthisbs.
  3. The desk checks the claim against independent sources and replies with the verdict — and a link back here for the receipts.

Checks summoned on X become permanent articles on this site, usually within a day.

Built on the Lenz API

This whole site is a small, open-source Python program (MIT license) that reads the public Lenz catalog through the lenz-io SDK and renders static HTML — no server, no database of its own. It doubles as a reference implementation for anyone building on the API.
